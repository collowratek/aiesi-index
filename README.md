# Heatmapa AI — AIESI Index

## Co to je
Streamlit aplikace pro vizualizaci AIESI indexu (AI readiness/adopce) pomocí heatmap a interaktivních grafů.

## Tech stack
- Python
- Streamlit
- Plotly (vizualizace)
- Pandas (data processing)
- AIESI metodologie

## Struktura
- `app/` — Streamlit aplikace
- `data/` — Datové soubory
- `docs/` — Dokumentace a metodologie

## Jak spustit
```bash
pip install -r requirements.txt
streamlit run app/main.py
```
